# AKYSBDRAWS

Offline-first tournament draw PWA. No backend, no database — runs entirely in the browser.

## Features
- Single Elimination
- Double Elimination
- Round Robin
- Swiss Format
- Visual bracket — click a player name to set them as winner
- Undo any result
- Live standings table (Round Robin & Swiss)
- Installable as PWA (works offline)

## Setup
```bash
npm install
npm run dev
```

## Build & Preview PWA
```bash
npm run build
npm run preview
```
